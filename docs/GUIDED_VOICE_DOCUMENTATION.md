# Guided Voice Documentation

The MVP supports typed notes, standard voice note demo transcription, guided interviews, transcript editing, AI improvement, quality scoring, and SpeechSynthesis read-back.

Production should add browser SpeechRecognition where supported, plus server-side audio transcription for reliable mobile and accessibility support. Original transcripts must remain separate from final notes for audit trail.

High-impact actions such as submit, self-certify, approve, send, delete, and invoice should require confirmation, especially when Accessibility Mode is enabled.
